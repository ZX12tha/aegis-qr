# 🛡️ AEGIS AI PORTAL
**Advanced AI-Powered QR Phishing Detection System**

AEGIS is a web-based cybersecurity tool designed to scan and analyze QR codes in real-time. It helps users identify potential "Quishing" (QR Phishing) threats by analyzing URLs before they are opened.

## 🚀 Key Features
- **Real-time Camera Scan:** Leverages OpenCV for fast QR code recognition.
- **Threat Intelligence:** Analyzes URLs to detect malicious patterns and suspicious domains.
- **Instant Status:** Provides visual (Safe/Threat) and audio feedback.
- **Containerized Deployment:** Optimized with Docker for high stability.

## 🛠️ Tech Stack
- **Language:** Python
- **UI Framework:** Streamlit
- **Computer Vision:** OpenCV, PyZbar
- **DevOps:** Docker, Hugging Face Spaces
